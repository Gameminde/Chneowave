Saving a Configuration File

You can save the device, channel, sample rates, triggers, and display settings from the latest acquisition to a configuration file. A DAQami configuration file is saved with the extension .amicfg.

To save a configuration file, select »Save Configuration and navigate to the folder where you want to save the configuration (.amicfg) file.

The first time you save a configuration to file, a dialog box opens for you to enter the name and select the location of the file.

Select Save As Configuration to save an existing configuration file with a different name.

Configuration files are saved by default to ..\My Documents\Measurement Computing\DAQami\.
