Opening a Configuration File

When you save a configuration, the device, channel, acquisition, and display settings are written to a .amicfg configuration file.

You can open a configuration file from the Getting Started dialog box and from the Gear menu.

From the Getting Started Dialog Box

Click the Open Configuration button and navigate to the .amicfg configuration file to open.

From the Gear Menu

Select »Open Configuration and navigate to the .amicfg configuration file to open.

When you open a configuration file, the DAQami main window updates with the device, channels, and display settings that are saved in the file.
