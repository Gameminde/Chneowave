Updating Display Settings in a Data File

When you acquire and log data to file, in addition to the data, all device, channel, sample rate, trigger, and display settings are saved in the file.

If you have a data file open and add, remove, or change the settings of displays, you can save these updated display settings to the data file. However, changes to device, sample rate, and trigger settings are not saved, and most changes to channel settings are saved to the open data file.

To update display settings and save them to a data file, complete the following steps:

  1. Create a data file by clicking to acquire and log data. The data file remains open when the acquisition stops.
  2. Or
  3. Open an existing data file.
  4. Add, remove, and edit the settings of displays as needed. You can also edit the names and trace colors of channels.
  5. Select »Save Display Settings to Data File.

The following changes are saved to the data file:

  • Any displays that were added to or removed from the acquisition.
  • Any changes to the visual and axes settings on displays.
  • Any changes to channel names and channel (trace) colors.

If you activate new channels and add them to a display, the channels themselves are not saved to the data file. When you open the updated data file, the channels are shown on the display, but they will not show any data when an acquisition runs.
