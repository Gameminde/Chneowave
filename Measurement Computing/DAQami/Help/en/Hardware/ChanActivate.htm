Activating Channels

Data is acquired from all channels that are selected, or activated, for the acquisition. Use the Configuration Panel to activate each channel. Complete the following steps to activate one or more channels:

  1. In the Device Panel, select a device with the channels to activate.
  2. Click on the Panel button to display the Configuration panel.
  3. Click on the Channels tab.
    Channel options are grouped by signal type.
  4. Select the Active checkbox for each channel you want to activate.

To remove a channel from the acquisition, clear its Active checkbox.

Data from each activated channel can be viewed on displays and logged when you run an acquisition.

Note: Acquired data is not displayed until you add the channel to a display.
