Add Network Device dialog box

The Add Network Device dialog box displays when you drag and drop the Network Device icon onto the Device Panel.

DAQami searches for devices connected to the network, and lists detected devices on this dialog box.

  • DAQami automatically detects network devices installed on a local network with DHCP enabled.
  • For devices installed on a local network without DHCP, you must enter the host name or IP address of the host network before DAQami can detect it.
  • For devices installed on a remote network, you must also enter the port number to forward.

Examples and available settings on the Add Network Device dialog for each detection type are listed in the following table.

Detection TypeSelect Automatic from the list to detect devices installed on the local network with DHCP enabled. This is the default (and most common) detection type, and no additional information is required to detect a device.
  • Network devices detected by DAQami are listed under Devices Found. Select a device and click OK to add the network device to the DAQami configuration.
  • If the device you want is not listed, click the Refresh Devices List icon to search again for network devices.
