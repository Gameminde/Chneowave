Adding a Device

You can add multiple USB, Bluetooth, and network devices to the Device Panel and to an acquisition. Only one instance of the DEMO-BOARD can be added to an acquisition at a time.

USB/Bluetooth® device, or the DEMO-BOARD

Click Add Device and drag and drop a USB or Bluetooth device , or the DEMO-BOARD , onto the Device Panel.

Repeat this step to add other devices to the acquisition.

After you add a device, the Configuration Panel automatically opens for you to configure device, channel, sample rate, and trigger settings.

Refreshing the Device List

If you connect or disconnect a device during the current acquisition session, click Add Device and then click to refresh the list of detected USB and Bluetooth devices.

Network device (Automatic or Manual detection)

Important: Make sure you have written down the MAC address of the network device you want to use.

The MAC address can be found on a label on the bottom of the device enclosure, or on the board itself for OEM models.

Complete the following steps to add a network device to a DAQami acquisition:

  1. Click Add Device and drag and drop the Network Device icon onto the Device Panel to open the Add Network Device dialog box, which lists detected network devices.
  2. Select one of the following settings from the Detection Type list:
    • Automatic (default) to detect a device installed on the local network with DHCP enabled.
      1. If the device is not listed under Devices Found, click icon to perform another search for network devices.
      2. Select the device – based on the MAC address – listed under Devices Found that you want to add to the acquisition.
    • Manual to detect a device installed on the local network with DHCP disabled, or installed on a remote network.
      1. Enter the host name or IP address of the host network router in the Address textbox.
      2. If the device is connected to a remote network, select the Remote Network checkbox and enter the port number to forward, for example, 54211, in the text box after the host name/address.
    Note: you cannot add a network device that is currently in use.
  3. Make sure the correct Connection Code is entered (if applicable). The default code is 0000. and click OK to add the network device to the Device Panel.

Repeat these steps to add other network devices to the acquisition.

After you add the device, the Configuration Panel automatically opens for you to configure device, channel, sample rate, and trigger settings.

Note: you cannot add a network device that is currently in use.

Removing a Device

To remove a device from the Device Panel and the current acquisition, click . A Warning dialog box prompts you to confirm deleting the device and all device, channel, sample rate, and trigger settings.
