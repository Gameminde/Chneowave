Display Overview

A DAQami display shows data acquired from selected input channels. You can add any combination of Scalar, Strip, and Block displays to a Display tab.

The DAQami Output display provides controls to generate and update output signals.

Input channel displays includes icons that provide access to display settings. You can also right-click on a display or on a channel added to a display to open a context menu with additional settings.

After you run an acquisition, you can drag channels between displays, and the acquired data is automatically added to the display.
