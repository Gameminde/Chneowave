Adding and Modifying Displays

When you create a new configuration, the display panel contains a blank display tab (Tab 0) with a Display Row placeholder for the first display to be added to the display tab.

Complete the following steps to add a display to a display tab on the Display Panel:

  1. Click the Add Display button to show the list of displays and also open the Display Panel if it is not currently showing.
  2. Click on a display and drag it onto the Display Panel.
  3. Repeat to add other displays above, below, or on either side of existing displays. When the target area on the Display Panel turns green, release the mouse button to add the display. Each display is automatically resized to fit in the row.

You can change the height and width of each display, customize display options, and drag a display to a different area of the Display Panel. You can also copy and move channels from one display to another.

Resizing, Moving, and Removing Displays

You can change the size and location of each display on the Display Panel, stack them horizontally and/or vertically, and remove a display from the display panel.

  • To resize a display, click and drag on the display bottom or right edge.
  • To move a display, click in the upper part of the display and drag it above, below, or adjacent to another display. When the target area on the Display Panel turns green, release the mouse button.
  • To remove a display, click the Remove Display icon on the display, or right-click on the display and select Remove Display from the context menu.
  • If you accidentally remove a display, add the display to the Display Panel again and then drag over the same channel(s) to the display. The display is automatically populated with the acquired data for each channel.

Each display has settings customizing its visual appearance. For more information go to the Configuring Display Settings topic.
