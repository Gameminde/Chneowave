DAQami Step-by-Step Guide

Select a device

  1. Click Add Device to display a menu of available devices.
  2. Select a device from the menu, drag the device onto the device panel, and release the mouse button to add the device.
  3. Repeat steps 1 and 2 to add other devices.

Configure the device

  1. For each device, click on the Channels tab and select the Active checkbox for each channel to sample or update.
  2. Change the input or output setting for each channel as needed.
  3. Set the sample rates for input channels on the Sample Rates tab.
  4. Set triggers used for all channels on the Triggers tab.

Add a display

  1. Click Add Display .
  2. For input channels, select a Scalar, Strip, or Block display.
  3. For output channels, select an Output display.
  4. Drag the display onto the Display Panel, and release the mouse button to add the display.
  5. Drag channels from the Active Channels list onto the display.
  6. Repeat steps 1 through 3 to add other displays and channels.

Log, view, and update data

  1. Click to log, view, and update data.
  2. Or
  3. Click to view and update data without logging.
  4. Click to stop acquiring data.
