Introducing DAQami Software

DAQami data acquisition companion software is used to acquire, view, log, and update analog, digital, and counter I/O data from supported Measurement Computing DAQ devices.

DAQami features a drag-and-drop interface for you to quickly and easily add devices, displays, and channels. You can acquire, view, and log data in four steps:

  1. Select one or more Measurement Computing DAQ devices.
  2. Configure device, channel, sample rate, and trigger settings.
  3. Add one or more displays to show the acquired data and/or to control output signals.
  4. Run the acquisition.

Acquired data is shown on the displays, and can also be logged to a data file.
