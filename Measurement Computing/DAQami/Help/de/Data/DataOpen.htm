Gespeicherte Daten öffnen

DAQami speichert Messdaten und Konfigurationseinstellungen in eine .tdms-Datei bei jedem Starten einer Erfassung mit .

Gehen Sie folgendermaßen vor, um eine Datendatei zu öffnen:

  1. Starten Sie DAQami und wählen Sie Gespeicherte Daten öffnen im Fenster Erste Schritte, oder wählen Sie »Gespeicherte Daten öffnen....

  2. Wenn eine Konfigurationsdatei geöffnet ist, werden Sie gefragt, ob Sie Änderungen speichern möchten.
  3. Wählen Sie eine DAQami Datendatei mit Gespeicherte Daten öffnen... aus.
  4. Da DAQami im Arbeitsspeicher nur maximal 1 Million Samples pro Kanal speichern kann, werden Sie aufgefordert einen Datenbereich auszuwählen, sofern die Datei mehr als 1 Million Samples pro Kanal enthält.

    Mit dem Rollbalken kann der Datenbereich, der geladen werden soll, ausgewählt werden. Alternativ kann ein Bereich über Absolute Zeit/Relative Zeit (s) ausgewählt werden.



    Tipp: Sie können im Erste-Schritte-Fenster auch eine Datei auswählen unter Zuletzt verwendete Datendateien .

Beim Öffnen gespeicherter Messdaten werden die zugehörigen Einstellungen für Gerät, Erfassungsrate, Trigger und Anzeigen in DAQami geladen. Alle Funktionen von DAQami können mit den geöffneten Daten verwendet werden. Anzeigen, Kanäle, Cursor und so weiter, können hinzugefügt werden.

Wenn Sie Anzeigen hinzufügen oder entfernen, Anzeige-Einstellungen ändern und/oder Farbe und Bezeichnung eines Kanals ändern, können Sie diese Einstellungen speichern mit »Anzeigeneinstellungen in der Datendatei speichern.

Sie können Daten neu erfassen, solange dasselbe Gerät noch verbunden ist. Wenn Sie Daten neu erfassen, wird eine neue Datendatei erstellt. Wählen Sie Überschreiben unter Logging-Optionen, wenn Sie die vorhandene Datei mit den neuen Daten überschreiben möchten.
