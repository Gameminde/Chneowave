Daten exportieren

Sie können bis zu 1 Million Messwerte pro Kanal in comma-separated values (.csv) Dateien exportieren über die Einstellungen unter Export-Optionen.

Falls Sie mehr als 1 Million Messwerte pro Kanal erfasst haben, werden die letzte Million Samples in die Dateien exportiert.

Öffnen Sie den Dialog » Daten exportieren... .

Wen Sie eine DAQami Datendatei exportieren, wird eine .csv-Datei generiert für jedes Gerät und jedes Eingangs-Subsystem – analog, digital und/oder Zähler. Jede Datei besitzt denselben Dateinamen mit einer Ergänzung für das Gerät, den Datentyp und den Erfassungszeitpunkt.

Wenn Sie beispielsweise eine Datei exportieren, die mit einem USB-2408-2AO erfasste analoge, digitale und Zählerdaten enthält und mit einem USB-1608GX erfasste digitale und Zählerdaten, dann werden beim Export für jedes Gerät und jedes Subsystem die folgenden fünf Dateien erzeugt:

Export-Optionen
Export nachKlicken Sie auf Durchsuchen... und legen Sie den Speicherort für die .csv-Dateien fest. Die Daten werden standardmäßig exportiert in das Verzeichnis \Users\<current user>\Dokumente\Measurement Computing\DAQami.
Optionen
  • Genauigkeit (Dezimalstellen): Geben Sie die Anzahl der Dezimalstellen ein, mit der die Daten exportiert werden sollen.
  • Relative Zeit verwenden: Markieren Sie diese Checkbox zum Export der Daten mit der relativen Zeit der Erfassung. Aktivieren Sie diese Checkbox nicht, wenn Sie die Daten mit der absoluten Zeit bei Erfassung (Computerzeit) exportieren möchten.
  • Bereich eingrenzen: Markieren Sie diese Checkbox, um nur Daten aus einem eingeschränkten Zeitbereich zu exportieren. Sie können den Bereich angeben in Absolute Zeit oder Relative Zeit (s). Jeder Bereich passt sich an die Änderungen eines anderen Bereiches an.
  • Dezimalformat: Klicken Sie auf den Pfeil abwärts und wählen Sie das Dezimalformat für den Export in eine .csv-Datei:
    • Windows-Einstellung: Verwenden Sie das Dezimaltrennzeichen, das unter Systemsteuerung»Region»Weitere Einstellungen...»Zahlen eingestellt ist.
    • Punkt: Benutzt den Punkt (.) als Dezimaltrennzeichen.
    • Komma: Benutzt das Komma (,) als Dezimaltrennzeichen.
  • Trennzeichen: Klicken Sie auf den Pfeil abwärts und wählen Sie das Zeichen, mit dem jede Spalte getrennt werden soll beim Export von Daten in eine .csv-Datei:
    • Windows-Einstellung: Verwendet das Listentrennzeichen, das unter Systemsteuerung»Region»Weitere Einstellungen...»Zahlen eingestellt ist, um jede Spalte der Daten zu trennen.
    • Tab: Verwendet den Tabulator zum Trennen der Spalten der Daten.
    • Semikolon: Verwendet das Semikolon (;) zum Trennen der Spalten der Daten.
    • Komma: Verwendet das Komma (,) zum Trennen der Spalten der Daten.
  • Werte in Anführungszeichen: Verwendet Anführungszeichen zur Kennzeichnung von Werten beim Export in eine .csv-Datei.

    Wenn diese Feld nicht markiert ist, wird kein Zeichen zur Kennzeichnung von Werten verwendet.
