Hinzufügen eines Cursors in einer Anzeige

Fügen Sie einen Cursor zur Datenanzeige in einem Linienschreiber oder Oszilloskop hinzu, um die Werte einzelner Datenpunkte anzuzeigen.

  • Sie können bis zu zwei Cursor zu einem Linienschreiber oder Oszilloskop hinzufügen.
  • Cursor können nur zu jeweils demselben Kanal einer Anzeige hinzugefügt werden.
  • Die Farbe des Cursor entspricht der Farbe des zugeordneten Kanals.
  • Cursor können einer Anzeige vor, während oder nach einer Erfassung hinzugefügt werden.

Gehen Sie folgendermaßen vor, um Cursor zu einer Anzeige hinzuzufügen:

  1. Wenn Daten gerade erfasst werden, klicken Sie auf die Feststell-Schaltfläche um in den Rückblick-Modus zu wechseln. Andernfalls gehen Sie zu Punkt 2.
  2. Klicken Sie auf die Cursor-Symbole (Cursor A oder Cursor B ).
    Ein Fadenkreuz wird der Anzeige des aktuell ausgewählten Kanals hinzugefügt. Die X-Koordinate zeigt entweder die Erfassungszeit (absolut oder relativ) oder die Nummer des Messwertes, entsprechend den Anzeige-Einstellungen für die X-Achse. Die Y-Koordinate zeigt den Messwert.
  3. Klicken Sie auf das andere Cursor-Symbol, um einen zweiten Cursor hinzuzufügen.
    Werden zwei Cursor hinzugefügt, wird zusätzlich die Differenz in Zeit und Amplitude zwischen den beiden Cursor angezeigt.
  4. Ziehen Sie das Fadenkreuz entlang der Anzeige, um die Werte jedes Datenpunktes anzuzeigen. Ein A oder B identifiziert den Cursor auf der Anzeige.
  5. Klicken Sie das Entsperren-Symbol , um den Rückblick-Modus zu verlassen und zur Anzeige der aktuell erfassten Messwerte zurückzukehren.

Das Bild zeigt zwei Cursor, die dem Kanal 0 auf dem Linienschreiber hinzugefügt wurden.

Klicken Sie erneut auf das Cursor-Symbol, wenn Sie Cursor aus der Anzeige entfernen möchten.

Um den Wert der Datenpunkte auf einem anderen Kanal anzuzeigen, klicken Sie auf das Symbol des Kanals. Die Cursor wechseln zu dem ausgewählten Kanal.

Skalierung und Cursor

Wenn Sie Cursor zu einem Linienschreiber oder Oszilloskop hinzufügen und dann die Anzeige so zoomen, dass die Cursorposition außerhalb der angezeigten Daten wandert, dann zeigen Pfeile (, ) die Position der Cursor relativ zu den angezeigten Daten an.
