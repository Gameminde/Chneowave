Übersicht der Anzeigen

Eine Anzeige in DAQami zeigt die Messwerte der ausgewählten Kanäle. Sie können eine beliebige Kombination aus Numerischen Anzeigen, Linienschreibern und Oszilloskopen einer Anzeige hinzufügen.

Die Ausgabe in DAQami ermöglicht Steuerung der Signalgenerierung und Signalausgabe.

Anzeigen für Eingangskanäle besitzen Symbole zur Einstellung der Anzeigen. Mit rechtem Mausklick auf eine Anzeige oder einen hinzugefügten Kanal wird ein Kontextmenü für zusätzliche Einstellungen geöffnet.

Nach Beendigung der Erfassung können Sie Kanäle zwischen den Anzeigen ziehen. Die erfassten Daten werden automatisch dargestellt.
