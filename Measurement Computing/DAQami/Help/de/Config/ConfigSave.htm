Eine Konfiguration speichern

Sie können die Einstellungen für Gerät, Kanal, Erfassungsraten, Trigger und Anzeigen der letzten Erfassung in einer Konfigurationsdatei speichern. Eine Konfigurationsdatei von DAQami wird mit der Erweiterung .amicfg gespeichert.

Wählen Sie »Konfiguration speichern um eine Konfigurationsdatei zu speichern und wechseln Sie zu dem Order, in dem Sie die Konfigurationdatei (.amicfg) speichern möchten.

Wenn Sie eine Konfiguration das erste Mal speichern, öffnet sich ein Dialogfeld, in dem Sie Namen und Speicherort der Datei festlegen können.

Wählen Sie Konfiguration speichern unter... um eine vorhandene Konfigurationsdatei unter einem anderen Namen zu speichern.

Der Standardpfad für Konfigurationsdateien ist ..\Dokumente\Measurement Computing\DAQami\.
