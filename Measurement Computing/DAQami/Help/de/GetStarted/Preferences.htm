DAQami Voreinstellungen

Im Fenster "Voreinstellungen" können die Standardeinstellungen für die Anzeigen gesetzt, die Tool-Tipps aktiviert und die Einstellungen für den automatischen Export der erfassten Messdaten in eine .csv-Datei gesetzt werden.

Öffnen Sie die »Voreinstellungen.

Voreinstellungen
Anzeige
AnzeigenschemaSetzt das Anzeigenschema für jede Anzeige, die hinzugefügt wurde. Wählen Sie eines der folgenden Anzeigeschemata:
  • Hell: X- und Y-Achsen, sowie Text erscheinen schwarz vor einem weißen Hintergrund.
  • Dunkel: X- und Y-Achsen, sowie Text erscheinen weiß vor einem blauen Hintergrund.
X-Achse AnzeigemodusSetzt die X-Achsen-Einheit für die Dauer der Erfassung, die in allen Anzeigen verwendet wird. Die Zeit wird angezeigt im Format Stunde:Minute:Sekunde:Millisekunde (HH:MM:SS:MS). Klicken Sie auf das Feld und wählen Sie einen der folgenden Modi aus:
  • Absolute Zeit: die X-Achse zeigt die Zeit, zu der die Messwerte erfasst wurden.
  • Relative Zeit: die X-Achse zeigt die Zeit seit Beginn der Erfassung.
  • Messwertzähler: die X-Achse zeigt die Nummern der erfassten Messwerte.
Hinweis: Sie können das Anzeigenschema für eine bestimmte Anzeige auswählen, indem Sie in den Anzeige-Einstellungen auf das Symbol klicken.
Allgemein
Tool-Tipps anzeigenTool-Tipps anzeigen oder verbergen.
Automatischer Export
Automatischer ExportClick on the down arrow and select one of the following modes:
  • Immer fragen: Sie werden nach jeder Erfassung aufgefordert, die Daten in eine .csv-Datei zu exportieren.
  • Nie exportieren: Der automatische Export ist deaktiviert. Wählen Sie »Daten exportieren, um die aktuelle Erfassung in einer .csv-Datei zu speichern.
  • Immer exportieren: Die Messdaten werden nach jeder Erfassung automatisch in eine .csv-Datei gespeichert.
GenauigkeitGeben Sie die Zahl der Dezimalstellen an, mit denen die Daten exportiert werden sollen.
Relative Zeit verwendenAktivieren Sie diese Checkbox, um die Daten mit der relativen Zeitinformation zur Erfassung zu exportieren. Aktivieren Sie diese Checkbox nicht, wenn Sie die Daten mit der absoluten Zeit bei Erfassung (Computerzeit) exportieren möchten.
DezimalformatKlicken Sie auf den Pfeil abwärts und wählen Sie das Dezimalformat aus für den Export in eine .csv-Datei:
  • Windows-Einstellung: Verwenden Sie das Dezimaltrennzeichen, das unter Systemsteuerung»Region»Weitere Einstellungen...»Zahlen eingestellt ist.
  • Punkt: Benutzt den Punkt (.) als Dezimaltrennzeichen.
  • Komma: Benutzt das Komma (,) als Dezimaltrennzeichen.
TrennzeichenKlicken Sie auf den Pfeil abwärts und wählen Sie das Zeichen aus, mit dem jede Spalte von Daten getrennt wird, für den Export in eine .csv-Datei:
  • Windows-Einstellung (Standard): Verwenden Sie das Listentrennzeichen, das unter Systemsteuerung»Region»Weitere Einstellungen...»Zahlen eingestellt ist, um jede Spalte der Daten zu trennen.
  • Tab: Verwendet den Tabulator zum Trennen der Spalten der Daten.
  • Semikolon: Verwendet das Semikolon (;) zum Trennen der Spalten der Daten.
  • Komma: Verwendet das Komma (,) zum Trennen der Spalten der Daten.
Werte in AnführungszeichenWählen Sie dieses Feld aus, um Werte mit Anführungzeichen zu kennzeichnen beim Export in eine .csv-Datei.

Wenn diese Feld nicht markiert ist, wird kein Zeichen zur Kennzeichnung von Werten verwendet.
Hinweis: Die Einstellungen für Automatischen Export werden in einer Evaluierungslizenz nach der 30 Tagen deaktiviert. Wenn Sie eine DAQami-Lizenz erwerben, werden Datenspeicherung und -export wieder freigeschalten.

Klicken Sie auf OK, wenn Sie die Änderungen in den Voreinstellungen speichern möchten.
