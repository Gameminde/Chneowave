Erste Schritte mit DAQami

Das Fenster Erste Schritte öffnet sich beim Start von DAQami. In diesem Fenster kann eine neue Konfiguration erstellt oder eine vorhandene Konfiguration oder gespeicherte Daten geöffnet werden.

Hinweis: Um dieses Fenster zu öffnen, während DAQami bereits gestartet ist, wählen Sie »Erste Schritte zeigen....

Erste-Schritte-Fenster
Neue KonfigurationErzeugt eine neue Konfigurationsdatei. Sie werden aufgefordert Manuell oder Automatisch im Fenster Neue Konfiguration: auszuwählen.
  • Mit Manuell können ein oder mehrere Geräte ausgewählt und Gerät, Kanal, Erfassungsrate, Trigger und Anzeige-Einstellungen konfiguriert werden.
  • Mit Automatisch wird eine DAQami-Konfiguration geöffnet mit bis zu vier aktivierten und zu einer Anzeige hinzugefügten Kanälen.

    Das USB- oder Bluetooth-Gerät, das zuerst verbunden wurde, wird zur Konfiguration hinzugefügt. Falls kein USB- oder Bluetooth-Gerät erkannt wird, wird das "Demo-Gerät" hinzugefügt.

Hinweis: Wenn Sie eine dieser Optionen aus dem Fenster Neue Konfiguration: wählen, werden Sie aufgefordert, die aktuelle Konfiguration zu speichern. Wenn Sie auf Nein klicken, wird die Konfiguration gelöscht.
Konfiguration öffnen...Mit Konfiguration öffnen wird eine vorhandene Konfiguration geöffnet. Wählen Sie die .amicfg-Konfigurationsdatei aus.
Gespeicherte Daten öffnen...Mit Gespeichert Daten öffnen wird eine vorhandene Datendatei geöffnet. Navigieren Sie in diesem Fenster zur .tdms-Datendatei, die geöffnet werden soll.

Sie können auch ältere .mcx und .mcd Datendateien öffnen, die mit DAQami V3.0 oder früher erstellt wurden.
Zuletzt verwendete KonfigurationenZeigt die zuletzt verwendeten Konfigurationsdateien an. Klicken Sie auf eine Datei, um die Konfiguration in DAQami zu laden.
Zuletzt verwendete DatendateienZeigt die zuletzt verwendeten Datendateien. Klicken Sie auf eine Datei, um sie in DAQami zu öffnen.
DAQami HilfeÖffnet diese Hilfe-Datei.
Schritt-für-Schritt AnleitungÖffnet die Schritt-für-Schritt Anleitung. Dieser Abschnitt beschreibt, wie ein Gerät hinzugefügt und konfiguriert, eine Anzeige hinzugefügt und Messdaten erfasst werden.
