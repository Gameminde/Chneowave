Tools-Menü (Zahnrad-Symbol)

Das Tools-Menü führt zu den Einstellungen für Dateien und Konfigurationen.

Ein Klick auf das Zahnrad-Symbol in der rechten oberen Ecke des DAQami-Fensters öffnet folgendes Menü:

Einstellungen
Neue KonfigurationEine neue Konfiguration erstellen und die aktuellen Einstellungen für Geräte, Kanäle, Erfassung und Anzeige löschen.
Konfiguration öffnen...Eine vorhandene Konfiguration aufrufen über das Fenster Konfiguration öffnen.
Konfiguration speichernSpeichert die Änderungen in der aktuellen Konfiguration. Falls die Konfiguration zuvor noch nicht gespeichert wurde, öffnet sich das Fenster Konfiguration speichern, um die Konfiguration in einer Datei zu speichern.
Konfiguration speichern unter...Speichert die aktuelle Konfiguration über das Fenster Konfiguration speichern. Verwenden Sie diese Option, wenn Sie die aktuelle Konfiguration in einer neuen Datei speichern möchten.
Anzeige-Einstellungen in der Datendatei speichernAktualisiert die Datendatei mit den Anzeigen, die hinzugefügt, gelöscht oder geändert wurden, seit die Datendatei geöffnet wurde. Mit Ausnahme von Kanalnamen und -farben werden Änderungen von Kanal, Erfassungsrate und Trigger nicht gespeichert.
Gespeicherte Daten öffnen...Öffnen Sie eine DAQami (.tdms) Datendatei oder eine ältere DAQami Datendatei (.mcx/.mcd – DAQami 3.0 und früher) über das Fenster Gespeicherte Daten öffnen. Sie werden vor der Auswahl einer Datendatei aufgefordert, die aktuelle Konfiguration zu speichern.
Daten exportieren...Exportiert die erfassten Daten in eine .csv-Datei in einem unter Export-Optionen angegebenen Verzeichnis.
Die Einstellungen können gemacht werden, sobald Daten zum Export verfügbar sind.
Logging-Optionen...Konfigurieren Sie Namen, Verzeichnispfad und den Logging-Modus (Datei Überschreiben oder Neue Datei erstellen) über die Logging-Optionen.
VoreinstellungenKonfigurieren Sie die Standardeinstellungen für Anzeige, Tool-Tipps und Automatischen Export (Umwandlung in .csv) über das Fenster Voreinstellungen.
Erste Schritte zeigen...Öffnen Sie das Erste Schritte-Fenster, um die Optionen für eine neue Konfiguration (Manuell oder Automatisch) einzustellen, eine vorhandene Konfiguration oder eine vorhandene Datendatei zu öffnen.
DAQami HilfeDiese HIlfe öffnen.
Über DAQamiIm Fenster Über DAQami werden Softwareversion, Treiberversionen und Kontaktinformationen angezeigt.
BeendenDAQami beenden.
Hinweis: Die Einstellungen unter "Daten exportieren" und "Logging-Optionen" werden innerhalb der Testlizenz nach 30 Tagen deaktiviert. Wenn Sie eine DAQami-Lizenz erwerben, werden Datenspeicherung und -export wieder freigeschalten.
