DAQami 详细教程

Select a device

  1. 点击添加设备 显示有效设备列表。
  2. 从菜单中选择设备，将设备拖至设备面板，松开鼠标添加设备。
  3. 重复步骤1和2添加其他设备。

配置设备

  1. 对于每个设备，点击通道选项，选择每个需要被使能的通道的激活复选框。
  2. 按需改变每个通道的输入或输出设置。
  3. 在采样率选项板中设置输入通道的采样率。
  4. 在触发选项板中设置所有通道的触发。

添加显示控件

  1. 点击添加显示控件 .
  2. 对于输入通道，选择标量、带状或块状显示。
  3. 对于输出通道，选择输出显示。
  4. 拖拽至显示面板，并松开鼠标即可添加到当前配置中。.
  5. 拖拽要激活的通道至显示面板中。
  6. 重复步骤1到3，添加其他的设备及通道。

记录，查看并更新数据

  1. 点击 记录，查看并更新数据
  2. 或
  3. 点击 查看、更新数据，但不记录。
  4. 点击停止数据采集。
