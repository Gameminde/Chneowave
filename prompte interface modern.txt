Mission Critique : Modernisation Design Interface CHNeoWave
Contexte et Objectifs
Projet : CHNeoWave - Système d'acquisition maritime
Problèmes identifiés : Interface surchargée, couleurs incohérentes, manque de fluidité, textes invisibles, workflow de calibration fragmenté, graphiques obsolètes
Objectif : Transformer l'interface en design moderne, fluide et cohérent selon les standards UI/UX 2025

CONTRAINTES ABSOLUES - NON NÉGOCIABLES
🚫 Interdictions Strictes
AUCUNE modification de la logique métier dans core/, hardware/, utils/

AUCUN changement des signatures de méthodes existantes

AUCUNE suppression de fonctionnalités actuelles

AUCUNE modification des workflows de données scientifiques

✅ Zone d'Intervention Autorisée
Fichiers QSS uniquement : gui/styles/

Layouts et widgets : gui/views/, gui/widgets/

Composants visuels : modifications esthétiques uniquement

Structure d'affichage : réorganisation visuelle sans impact fonctionnel

PROBLÈMES À RÉSOUDRE PRIORITAIREMENT

Solution requise :

css

4. Fluidité et Animations Professionnelles

css

Loading spinners modernes pendant traitements

5. Hiérarchie Visuelle et Espacement
Problème : Interface surchargée, pas de respiration

Solution requise :

Espacement Golden Ratio : 8px, 13px, 21px, 34px, 55px

Typographie hiérarchisée : H1(24px), H2(20px), H3(16px), Body(14px)

Ombres portées subtiles : box-shadow: 0 2px 8px rgba(0,0,0,0.08)

Bordures arrondies : border-radius: 8px pour modernité
